# Data Analysis in R: Students & Fire Dataset

## What
This project is an R-based data analysis notebook that explores two datasets:
- Student performance data (2020–2022)
- Fire incident dataset with missing values

It includes data cleaning, transformation, visualization, ranking, and statistical analysis using tidyverse tools.

---

## How
The analysis is performed using R with libraries such as `dplyr`, `tidyr`, and `ggplot2`.

### Student Dataset
- Reshaped data from wide to long format
- Visualized student marks across years
- Calculated total marks and ranked students
- Identified performance trends

### Fire Dataset
- Detected missing values and outliers
- Applied local median (for temperature) and mode (for confidence) imputation
- Compared original and imputed datasets
- Measured deviation to evaluate imputation quality

---

## Why
The goal of this project is to demonstrate practical data analysis skills, including:
- Cleaning and preparing real-world datasets
- Handling missing and inconsistent data
- Extracting meaningful insights through visualization
- Applying statistical techniques for better decision-making

This project reflects real-world scenarios where incomplete data must be processed effectively to ensure accurate analysis.
